# Ventoxia Beats 🎵

**Ventoxia Beats** is a web-based music maker built with HTML, JavaScript, and Supabase. Users can **sign up, log in, and play musical notes directly in the browser**. It features a simple sequencer and interactive keys for creating beats and melodies.

---

## 🚀 Features

- User authentication via **Supabase** (Sign Up / Login / Logout)
- Play musical notes using the **Web Audio API**
- Interactive **on-screen keyboard**
- Simple **8-step sequencer** to create and play sequences
- Responsive and browser-friendly interface
